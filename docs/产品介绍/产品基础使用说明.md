# 产品基础使用说明

## 程序运行

1. 进入项目根目录，找到 TronSensingShow.exe 文件，双击即可启动程序。
   <img width="642" height="62" alt="image" src="https://github.com/user-attachments/assets/e610c3fd-16ca-4bb6-b202-b7e61d7ed528" />


3. 若双击后无响应 / 无法启动，请将 log 目录下的日志文件提供给开发人员排查。

## 交互方式
支持鼠标点击、屏幕触摸两种方式，对界面控件进行操作交互。

## 快捷键说明
F2：切换至下一页（按 Pages.xml 配置顺序向下翻页）
F3：返回首页（快速跳转到 HomePage 页面）

## 关闭与最小化
1. 正常关闭：使用快捷键 Alt + F4 关闭程序
2. 最小化到桌面：使用 Win + D 快速返回桌面（程序后台继续运行）
3. 异常卡死处理：打开任务管理器，手动结束 TronSensingShow.exe 进程
